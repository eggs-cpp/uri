**Eggs.URI**
==================

## Introduction ##

**Eggs.URI** is a **C++11/14** URI library, as defined by
[RFC3986](http://www.rfc-editor.org/rfc/rfc3986.txt
"Uniform Resource Identifier (URI): Generic Syntax").
